BrowseMate

AI chat on any webpage — OpenAI, Gemini & Hugging Face. Your keys stay on your device.

Quick start

  • Install BrowseMate from the Chrome Web Store.
  • Click the BrowseMate icon → API Keys and add at least one key (OpenAI, Google Gemini, and/or Hugging Face).
  • Use the floating bubble on normal websites to open the chat panel.
  • Switch provider and model from the chat header.
