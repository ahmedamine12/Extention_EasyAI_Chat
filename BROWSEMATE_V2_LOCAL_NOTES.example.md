# Local notes template (tracked in Git)

**This file is safe to commit.** It only explains the workflow.

## Private file (not committed)

1. Copy **this folder’s** template content into a new file named **`BROWSEMATE_V2_LOCAL_NOTES.md`** (same directory as `README.md`).
2. That exact filename is listed in **`.gitignore`** — Git will not track it. Use it for:
   - quick **V2 vs old version** reminders;
   - the **comparison table**;
   - **LinkedIn / social** drafts (keep those out of `BROWSEMATE_V2_MARKETING.md`).

3. **Chrome Web Store** copy stays only in **`BROWSEMATE_V2_MARKETING.md`** (public).

If you cloned the repo fresh, create `BROWSEMATE_V2_LOCAL_NOTES.md` yourself (see gitignored pattern). You can paste sections from an older machine or rebuild from `CHANGELOG.md` + this checklist.
